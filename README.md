## Overview

## Featured Work

### Client Projects

**Payment Reconciliation Dashboard** | Bonita Cafe & De La Casa
- Daily reconciliation system processing multiple payment sources (Lightspeed POS, Zettle, Vipps, Givn, Wolt)
- Tech Stack: Next.js, React, Tailwind CSS, JWT Authentication
- Status: Production (Private repository)

**Receipt Management System** | Bonita Cafe
- Customer-facing receipt viewing application with clean UI design
- Tech Stack: Next.js, React, React Hooks
- [Live Demo](https://receipts.bonitahandel.no/) | Private repository

**Numa Booking System** | De La Casa
- Hotel breakfast tracking and verification system
- Tech Stack: Next.js, React, TypeScript
- Status: In development (Private repository)

### Academic Projects

**Holidaze - Accommodation Booking Platform**
- Full-featured booking platform with dual-role functionality
- Tech Stack: React, TypeScript, Tailwind CSS, API Integration
- [Live Demo](https://rosarioba.github.io/holidaze-booking-app/) | [Code](https://github.com/RosarioBA/holidaze-booking-app)

**BidLeaf - Auction Platform**
- Interactive auction website with real-time bidding
- Tech Stack: Vanilla JavaScript, Tailwind CSS, Vite, REST API
- [Live Demo](https://bidleaf.netlify.app/) | [Code](https://github.com/RosarioBA/bidleaf-sp2)
